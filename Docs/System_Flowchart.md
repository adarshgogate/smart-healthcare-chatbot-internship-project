# 🔄 System Flowchart

![System Flowchart](screenshots/system_flow.png)

**Patient Flow:**
1. Patient enters symptoms.
2. NLP extracts symptoms → ML model predicts condition.
3. Chatbot provides preliminary advice.
4. Severe cases escalate to doctor.

**Doctor Flow:**
1. Doctor logs in.
2. Reviews patient history + AI suggestions.
3. Confirms/overrides diagnosis.
4. Manages appointments + reminders.
